# AntibiAutomatismes

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Français

Générateur de sujets et d'exercices de mathématiques inspiré de la méthode **Antibi**, conçu pour produire rapidement des feuilles adaptées à l'entraînement des automatismes.

### Aperçu

![Interface AntibiAutomatismes](Exemples/interface.png)

### Contenu principal

- `main.py` : point d'entrée principal
- `generators.py` : génération des exercices
- `latex_builder.py` : construction LaTeX
- `preset.json`, `preset_train.json` : configurations d'exemple
- `requirements.txt` : dépendance minimale (`sympy>=1.12`)

### Utilisation

```bash
git clone https://github.com/philipperackette/AntibiAutomatismes.git
cd AntibiAutomatismes
pip install -r requirements.txt
python main.py
```

### Public visé

- enseignants de mathématiques,
- préparation de fiches d'entraînement,
- expérimentation pédagogique autour des automatismes.

## English

Mathematics worksheet and exercise generator inspired by the **Antibi** approach, designed to quickly produce material focused on procedural fluency and practice.

### Preview

![Interface AntibiAutomatismes](Exemples/interface.png)

### Main contents

- `main.py`: main entry point
- `generators.py`: exercise generation logic
- `latex_builder.py`: LaTeX builder
- `preset.json`, `preset_train.json`: example configurations
- `requirements.txt`: minimal dependency file (`sympy>=1.12`)

### Usage

```bash
git clone https://github.com/philipperackette/AntibiAutomatismes.git
cd AntibiAutomatismes
pip install -r requirements.txt
python main.py
```

### Intended audience

- mathematics teachers,
- worksheet generation workflows,
- educational experimentation around core skills training.
